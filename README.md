# Java Assignment 2: Pet Selection with Radio Buttons

This is a simple Java Swing desktop application that demonstrates the use of **JRadioButton**, **ButtonGroup**, **JLabel** for displaying images, and **JOptionPane** for showing messages.
Assignment Requirements (as given)
- Create an application with **five radio buttons** to choose a pet: Bird, Cat, Dog, Rabbit, Pig.
- Display the selected pet's image in the window.
- Show a message box (JOptionPane) confirming the selection.
- Upload the project to GitHub and share the repository URL.
- Provide screenshots and source code for verification.
 Features Implemented
- 5 radio buttons in a vertical panel with a titled border ("Choose your pet").
- Pig is pre-selected by default.
- Image updates instantly when a radio button is selected.
- Image scaling if too large (keeps aspect ratio).
- JOptionPane message box pops up on every selection.
- Error handling: Shows "Image not found or invalid" message if image fails to load.
- Clean layout using BorderLayout + GridLayout.
Technologies Used
- Java (JDK 8 or higher)
- Swing (javax.swing.*)
- AWT (java.awt.* for layout and images)
